Question Type Summary

All supported question types with their use cases

Text-Based Questions

  • Open Text: Detailed feedback and comments
  • Short Text: Single-line responses
  • Email: Email address collection
  • Phone: Phone number collection

Choice-Based Questions

  • Single Choice: One selection from options
  • Multiple Choice: Multiple selections allowed
  • Dropdown: Space-efficient selection
  • Yes/No: Binary choice questions

Rating & Scale Questions

  • Star Rating: 1-5 star visual rating
  • NPS Scale: 0-10 loyalty measurement
  • Likert Scale: Agreement scale
  • Slider: Continuous value selection

Advanced Questions

  • Matrix: Multiple questions, same options
  • Ranking: Order items by preference
  • Date/Time: Calendar and time pickers
  • File Upload: Document/image upload
